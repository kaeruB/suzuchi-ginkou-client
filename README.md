#Suzuchi Ginkou - Client
Track money borrowed by your partner.

## About
* Next.js
* SASS
* TypeScript

## Getting Started

First, build the application:
```bash
npm install
# or
yarn
```


Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
